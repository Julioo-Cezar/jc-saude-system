# 🏥 JC Saúde System

Sistema de gerenciamento para clínica médica desenvolvido para fins de estudo, aprendizado e composição de portfólio.

## 📋 Sobre o Projeto

O JC Saúde System é uma aplicação web que simula o ambiente de uma clínica médica, permitindo o gerenciamento de pacientes, médicos, consultas e relatórios.

O projeto está sendo desenvolvido com foco em aprendizado prático e aplicação de tecnologias utilizadas pelo mercado de desenvolvimento web.

## 🎯 Objetivos de Aprendizado

* Desenvolvimento Front-end
* Desenvolvimento Back-end
* Integração com banco de dados
* Controle de versão com Git e GitHub
* Boas práticas de organização de projetos
* Integração com APIs externas
